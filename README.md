# cat-filter

To test: ```python -m test.TransformerTest```

To run:
 - as local python project:
    ```
    pip install -r requirements.txt
    python main.py
    ```

 - as docker container:
    ```
    docker run -e API_HOST=http://agl-developer-test.azurewebsites.net arieszhang/cat-filter:latest
    ```
